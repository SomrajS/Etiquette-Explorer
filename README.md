# 🎮 Etiquette Explorer

An interactive 2D educational game developed using **Unity 2022.3 LTS** and **C#** to help children learn social etiquette through engaging, scenario-based gameplay. Players make decisions in everyday situations and receive instant feedback, encouraging positive behavior through gamification.

---

## 📌 Features

* 🏠 Interactive scenarios set in **Home**, **School**, and **Park**
* 💬 NPC-based dialogue system
* ✅ Decision-based gameplay with real-time feedback
* 🏆 Performance-based reward and scoring system
* 📈 Player progress tracking
* 🔄 Smooth scene transitions
* 💻 Fully offline educational experience

---

## 🛠️ Tech Stack

* **Game Engine:** Unity 2022.3 LTS
* **Programming Language:** C#
* **Paradigm:** Object-Oriented Programming (OOP)

---

## 🎯 Learning Objectives

* Teach children good manners and social etiquette
* Improve decision-making skills
* Encourage positive social behavior through interactive learning
* Make learning fun and engaging using gamification

---

## 🕹️ Gameplay

Players explore different real-life environments and interact with characters. Each scenario presents multiple choices, and the player must select the most appropriate response. Correct decisions earn points and rewards, while immediate feedback helps reinforce proper etiquette.

---

## 📂 Core Modules

* **Dialogue Manager** – Manages NPC conversations and player interactions.
* **Decision Evaluator** – Evaluates player choices and provides feedback.
* **Progress Tracker** – Records player progress and completed scenarios.
* **Reward System** – Awards points based on correct decisions.
* **Scene Manager** – Handles navigation between game environments.

---

## 🚀 Getting Started

### Prerequisites

* Unity Hub
* Unity 2022.3 LTS
* Visual Studio (recommended)

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/Etiquette-Explorer.git
   ```

2. Open the project using **Unity Hub**.

3. Load the project and press **Play** in the Unity Editor.
